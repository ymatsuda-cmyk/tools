アプリ概要

個人間の貸付金額・返済計画・返済実績を管理するシングルページアプリケーション（SPA）。データはGoogle Sheetsに保存し、GitHub Pagesで公開できる。

対象ユーザー

個人間でお金の貸し借りを管理したい人

公開方式

GitHub Pages（静的HTMLファイル1枚）

データ保存先

Google Sheets（GAS Web App経由）

通知機能

毎月1日にLINEグループ／個人へ自動送信

GAS API仕様
エンドポイント

GAS Web App URL（デプロイごとに発行）にクエリパラメータで action を指定して呼び出す。

GET {GAS_URL}?action=getLoans GET {GAS_URL}?action=getRepayments&loanId=L1234567890 GET {GAS_URL}?action=addLoan&name=田中&amount=200000&date=2025-05-01&monthly=20000 GET {GAS_URL}?action=addRepayment&loanId=L123&date=2025-05-01&amount=20000
アクション一覧
getLoans GET
パラメータ：なし
レスポンス：[{id, name, amount, date, monthly, memo}, ...]
addLoan WRITE
パラメータ：name, amount, date, monthly?, memo?
レスポンス：{id: 'L...'}
getRepayments GET
パラメータ：loanId
レスポンス：[{id, loanId, date, amount, memo}, ...]
addRepayment WRITE
パラメータ：loanId, date, amount, memo?
レスポンス：{id: 'R...'}
GASはCORSを自動で許可するためプロキシ不要。ただしリダイレクト（302）が発生するため fetch 時に redirect: 'follow' を指定すること。
🧮
ビジネスロジック
主要計算関数
getRemain(loan)
amount - Σrepayments.amount で残高を算出。返済履歴が0件なら元本をそのまま返す。
getPct(loan)
Math.min(100, round(paid/amount×100))。返済率を0〜100の整数で返す。超過登録がある場合も100%上限でキャップ。
calcEnd(loan)
残高 ≦ 0 → '完済済み'。monthly未設定 → '—'。それ以外は ceil(remain/monthly) ヶ月後の年月を YYYY年M月 形式で返す。
getStatus(loan)
残高・返済率からステータスバッジ（完済 / 要確認 / 返済中 / 順調）とCSSクラスを返す。
完済予定月の算出式
// 残高 ÷ 月額で必要月数を切り上げ計算 const months = Math.ceil(remain / monthly); const d = new Date(); d.setMonth(d.getMonth() + months); return `${d.getFullYear()}年${d.getMonth()+1}月`;
利子計算は含まない。元本÷月額の単純計算のみ。

Google Sheets / GAS 連携
設定項目値
GAS実行ユーザー自分（オーナー）
アクセス権限全員（匿名アクセス可）
デプロイ種別ウェブアプリ
URL保存先localStorage['gas_url']（端末ごとに設定が必要）
GAS側の主要関数
doGet(e) / doPost(e)
両メソッドともに handleReq(e) に委譲。LINE Webhookイベントの場合は events[0].source.groupId を検出してスクリプトプロパティに保存する。
handleReq(e)
action パラメータで分岐し、各CRUD関数を呼び出す。エラー時は {error: message} を返す。
setMonthlyTrigger()
既存の sendMonthlyReport トリガーを全削除してから新規作成（重複防止）。毎月1日 8:00 に設定。
GAS Web App URLはコードに直書きしないこと。GitHubに公開する際にURLが漏洩するリスクがある。localStorageに保存するか、URLを知っている人のみがアクセスできる運用とする。

LINE通知仕様
月次レポートの送信仕様
項目内容
実行タイミング毎月1日 8:00（GASタイムトリガー）
対象貸付残高 > 0 の貸付のみ
集計期間前月（実行月 - 1ヶ月）の返済データ
送信先スクリプトプロパティ LINE_GROUP_ID（未設定時は LINE_USER_ID）
APILINE Messaging API /v2/bot/message/push
ステータス判定（前月返済）
✅前月返済額 ≥ 月額返済予定予定通り返済済み
⚠️前月返済額 > 0（月額未満）一部返済
❌前月返済額 = 0未返済
認証情報の管理方法
プロパティ名取得方法設定方法
LINE_TOKENLINE Developersでチャネル発行setLineConfig() を実行して保存
LINE_USER_IDLINE Developersのチャネル基本設定setLineConfig() を実行して保存
LINE_GROUP_IDLINE Webhookイベントから自動取得ボットをグループに追加→メッセージ送信で自動保存
トークン等はコードに直書きせず、すべてスクリプトプロパティに保存すること（GASエディタ：プロジェクトの設定 → スクリプトプロパティ で確認可能）。
