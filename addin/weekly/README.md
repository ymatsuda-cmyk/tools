# 週次状況 Officeアドイン

「営業報告」シートと「wbs」シートから、選択した報告期間（**月〜金**）の実施状況を
**取引先別**に表示し、「報告」シートへタンキング（蓄積）するアドインです。

## 機能

- **週セレクタ**: ◀▶で前週・翌週へ移動。「今週」ボタン、日付ピッカーで任意の週へジャンプ
- **取引先別表示**: 取引先ごとに 件名／担当者／状態／状況（週次要約）をカード表示
- **ソースバッジ**: 営業報告（青）／WBS（緑）の由来を表示。ソース・取引先でフィルタ可能
- **報告シートへ自動保存**: 表示した週の内容を「報告」シートへ書き込み。
  **同一報告期間は洗い替え**（既存行を削除して書き直し）するため重複しません

## 抽出ロジック

状況テキストのうち「`7/8 …`」「`07/02 …`」「`2026/07/09 …`」のように
**日付で始まる行**をブロックの先頭とみなし、次の日付行までを1ブロックとして扱います。
選択した週（月〜金）に日付が該当するブロックだけを抽出します（新しい日付順）。
年のない `m/d` は報告週から年を推定します（年末年始も正しく判定）。

| ソース | 抽出元 | 取引先 | 件名 |
|---|---|---|---|
| 営業報告 | P列(進捗状況)・V列(商談状況)・W列(確認状況) | B列 | O列(問合せ・提案内容) |
| wbs | O列(備考)の日付付き記述 ＋ 実績開始日/実績終了日（「m/d 着手」「m/d 完了」として表示） | B列(小分類) | Z列(task) |

## 報告シートの列構成

| 列 | 内容 |
|---|---|
| A | 週開始日（月曜） |
| B | 週終了日（金曜） |
| C | ソース（営業報告／WBS） |
| D | 取引先 |
| E | 件名 |
| F | 担当者 |
| G | 状態 |
| H | 状況（週次要約） |
| I | 登録日時 |

シートが無ければ自動作成します。

## 設定（weekly.js 冒頭）

| 定数 | 内容 | 既定値 |
|---|---|---|
| `EXCLUDE_SUBCATS` | wbs 小分類の除外一覧（社内系など） | 事業部・経営・経理・総務・改善・引継ぎ・# |
| `EXCLUDE_CATEGORIES` | wbs 大分類の除外一覧 | （なし） |
| `CLIENT_ALIAS` | wbs 小分類 → 営業報告の取引先名への名寄せ | 柿本 → kakimoto arms |
| `INCLUDE_WEEKEND` | 土日の記述も同じ週に含めるか | false |

## デプロイ

1. `weekly/` フォルダを GitHub Pages の `tools/addin/weekly/` へ配置
2. 更新した `menu.json` を `tools/common/menu.json` へ上書き
   （「ツール」セクションに `id: weekly` を追加済み。カンバン・営業報告のスライドメニューにも自動で表示されます）
3. 単体のアドインとして使う場合は `manifest.xml` をサイドロード
   （既存のカンバン／営業報告アドインのメニューから開く場合はマニフェスト追加は不要）

## 注意

- Excel外（ブラウザ）で開くとDEMOモードで起動し、報告シートへの保存は行いません
- 報告シートへの保存は週の表示・切替のたびに自動実行されます
