# Excel アドイン手動追加手順

## デスクトップ版Excel（Windows）での手動追加

1. **Excelファイルを開く**
2. **ファイル** → **オプション** → **アドイン**
3. **管理**: **COMアドイン** → **設定**
4. **開発者用**の場合：**その他**選択
5. **マニフェストファイルから追加**:
   - **参照** で manifest.xml を選択
   - または **URL指定**: `https://ymatsuda-cmyk.github.io/tools/addin/kanban/manifest.xml`

## Web版Excel（ブラウザ）での確認

1. **Excel Online** を開く
2. **挿入** → **アドイン**  
3. **組織** タブを確認
4. **Excel Kanban** が表示されるか確認

## トラブルシューティング

### リボンにボタンが表示されない場合
- Excel再起動
- アドインの有効化確認： **ファイル** → **オプション** → **アドイン** → **無効なアドイン**確認
- キャッシュクリア（デスクトップ版）

### エラーが発生する場合  
- **開発者ツール**(F12)でコンソールエラー確認
- **信頼できる場所**の設定確認

## 確認項目

- [ ] アドインが**有効**になっている
- [ ] **ホーム**リボンタブを確認
- [ ] **カンバン**グループが表示される
- [ ] **カンバン表示**ボタンがある
