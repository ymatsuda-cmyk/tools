# バグ管理アドイン

「バグ」シート(2行目=項目名、3行目=入力例、4行目以降=データ)を、
Webアプリ／Officeアドイン両方で **一覧** と **カンバン** で表示・編集できます。

## ファイル構成
- index.html / styles.css / app.js : Webアプリ本体
- manifest.xml : Officeアドインマニフェスト
- assets/ : アイコン (16/32/80px)

## デプロイ手順 (GitHub Pages)
1. GitHub に新規リポジトリ作成 (例: `bug-tracker`) し、本フォルダの中身を push。
2. リポジトリ Settings → Pages → Branch を `main` / root に設定して保存。
3. `https://<ユーザ名>.github.io/<リポジトリ名>/` で公開される。

## マニフェストの編集
manifest.xml 内の以下を実際の値に置換してください。
- `<YOUR_GH_USER>` : GitHubユーザ名
- `<REPO>` : リポジトリ名
- `<Id>` : 新規GUID

## アドインのサイドロード (Excelデスクトップ)
1. 共有フォルダを「信頼されたアドインカタログ」に登録 (Excel→ファイル→オプション→トラスト センター→トラスト センターの設定→信頼できるアドインカタログ)。
2. その共有フォルダに manifest.xml を置く。
3. Excel→挿入→アドイン→個人用アドイン→共有フォルダー→「バグ管理」を選択。

Excel for the Web の場合は「アドインのアップロード」から manifest.xml を直接アップロード。

## 動作モード
- Excel内: バグシートを直接読み書き。保存でB〜V列上書き、D列(更新日)は本日に自動更新。
- ブラウザ単体: デモデータ表示。レイアウト確認用。
