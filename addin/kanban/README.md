# Excel Kanban アドイン インストールガイド

## 概要
Excelのデータをカンバンボード形式で表示・管理するアドインです。

## システム要件
- Microsoft Excel 2016以降（デスクトップ版）
- Windows 10/11
- インターネット接続（アドインの読み込みに必要）

## インストール方法

### 方法1: 自動インストール（推奨）

**日本語版 (文字化けしない版)**:
```powershell
# PowerShellを管理者権限で起動
.\install-ja.ps1
```

**English版**:
```powershell
# Run PowerShell as Administrator
.\install.ps1
```

### 方法2: 手動インストール
1. **開発者タブの有効化**:
   - Excel → ファイル → オプション → リボンのユーザー設定
   - 「開発」にチェックを入れる

2. **アドインの追加**:
   - Excel → 開発タブ → アドイン → Office アドイン
   - 「マニフェストのアップロード」をクリック
   - `manifest.xml` を選択

## 使用方法
1. Excelを起動
2. **ホーム**タブの**カンバン**グループ → **カンバン表示**ボタンをクリック
3. 右側のタスクペインにカンバンボードが表示されます

## 動作環境

- Microsoft Excel 2016 以降 (Office 365、Excel 2019、Excel 2021対応)
- Windows 10/11
- Internet Explorer 11 または Edge WebView2

## 機能
- ✅ 未着手・対応中・完了の3列表示
- ⭐ 優先作業の★マーク機能
- 🔍 担当者・分類・期間でのフィルタリング
- 📝 右クリックで備考編集
- 🔄 ドラッグ&ドロップでステータス変更
- 💾 フィルタ・サイズ設定の自動保存

## データ要件
- シート名: **wbs**
- 11行目以降にタスクデータ
- 列構成:
  - A列: 分類
  - N列: 担当者  
  - O列: 備考
  - P列: 開始日
  - Q列: 終了日
  - R列: 実績開始日
  - S列: 実績終了日
  - Y列: タイトル
  - Z列: ID

## アンインストール
```powershell
.\uninstall.ps1
```

## アンインストール方法

### 方法1: 自動アンインストール（推奨）

**日本語版**:
```powershell
.\uninstall-ja.ps1
```

**English版**:
```powershell
.\uninstall.ps1
```

### 方法2: 手動アンインストール
1. Excel → ファイル → オプション → アドイン
2. 「Excel Kanban」を選択して削除
3. Excelを再起動

## トラブルシューティング

### アドインが表示されない
1. Excelを完全に終了して再起動
2. ファイル → オプション → アドイン → 「Excel Kanban」が表示されるか確認

### ボタンが見つからない
- ホームタブの右端に「カンバン」グループがあるか確認
- リボンの幅が狭い場合は省略されている可能性があります

### エラーが発生する
1. インターネット接続を確認
2. シート「wbs」が存在するか確認  
3. データが11行目以降にあるか確認

## タスク追加機能（rev_20260710_taskadd）
ヘッダーの「＋タスク追加」ボタンから、wbsシートへ行挿入でタスクを登録できます。

**使い方**
1. ボタン押下で入力画面が開き、wbsシートが自動的にアクティブになります（「追加したい行を選択してください。」）。
2. wbsシート上で挿入したい行をクリックして選択。
3. 大分類・小分類・タスク名（必須）・担当者・予定開始日・予定終了日を入力し「OK（選択行に挿入）」。

**動作仕様**
- 選択行は名前定義「タスク範囲」内であることを検証（範囲外はエラー表示）。
- 選択行の位置に行挿入し、**T〜FY列の数式は隣接行（上、先頭行なら下）からコピー**して埋めます。
- 値の書込み: A=大分類 / B=小分類 / E=タスク名 / N=担当者 / P=予定開始日 / Q=予定終了日。
- 大分類で「**受注**」を選ぶと、小分類は**営業報告シートの状態が受注・受託中の案件番号**のセレクトに切り替わります（営業報告アドインの受託中タブ「WBS状況」はこの紐付けで集計されます）。
- Office環境では window.confirm が使えないため、アプリ内ダイアログを使用しています。

**事前準備**: wbsシートの行挿入可能な範囲に名前「タスク範囲」を定義してください（例: A11:FY400 を選択→名前ボックスに「タスク範囲」）。

## サブタスクカンバン（rev_20260710_subtask）
タスクの備考編集モーダル内に、備考テキストの下へサブタスク用の3レーンカンバンを追加しました。

- 備考テキストの行頭記号でサブタスクの状態を表します: □ 未着手 / ◎ 対応中 / ■ 完了
- カードをドラッグしてレーン間を移動でき、移動内容は備考テキストへ即時反映されます。
- 「＋追加」または入力欄でEnterすると □（未着手）のサブタスクが備考末尾に追加されます。
- 備考テキストを直接編集すると、カンバンも自動的に再描画されます（双方向同期）。
- 保存（O列書込み）時は従来どおり折り返し無効・行高さ固定です。
