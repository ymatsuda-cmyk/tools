📋 Scenario-tracker

Excel シナリオ一覧の作成状況・バグ影響を管理する Office アドイン

📥 インストール手順

  1. manifest.xml をダウンロードし、YOUR_GITHUB_USERNAME を自分のユーザー名に書き換える
  2. Excel を開き、挿入 → アドイン → アドインのアップロード
  3. 書き換えた manifest.xml を選択
  4. シナリオ一覧の Excel ファイルを開き、リボンの 「シナリオ一覧」 をクリック
manifest.xml をダウンロード ▶ デモを見る

📊 機能

  • 作成状況ビュー：5レーンのカンバンボード（未着手 / バグ保留 / 対応中 / 完了（条件付き）/ 完了）
  • バグ影響ビュー：バグIDごとに影響シナリオを一覧表示
  • 解消確認：チェックでExcelの該当セルに （済） を自動付与
  • フィルター：フェーズ・ブランド・シート・テキスト検索
  • デモモード：Excel未接続時は自動でデモデータ表示
