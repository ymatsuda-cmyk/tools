# AIニュース速報アプリ 仕様書

## 📋 概要

### アプリ名
**AIニュース速報** (AI News Tracker)

### 目的
AI関連ニュースの収集・整理・閲覧・保存を効率的に行うためのWebアプリケーション

### 対象デバイス
- モバイル（タッチ操作最適化）
- デスクトップ（マウス操作対応）
- PWA対応（アプリ化可能）

## 🛠 技術仕様

### フロントエンド
- **言語**: HTML5, CSS3, JavaScript (ES5)
- **フレームワーク**: なし（Pure JavaScript）
- **フォント**: Noto Sans JP, DM Mono
- **レスポンシブ**: CSS Flexbox, CSS Grid

### バックエンド
- **Google Apps Script (GAS)** - API サーバー
- **Google Sheets** - データ永続化
- **localStorage** - ローカルキャッシュ

### PWA機能
- **Web App Manifest**: あり
- **Service Worker**: なし
- **オフライン対応**: localStorage ベース

## 📊 データ構造

### メインデータフォーマット (JSON)
```json
{
  "date": "2026-05-04",           // 保存日付 (YYYY-MM-DD)
  "fetchedFor": "2026-05-03",     // 取得対象日付
  "summary": "今日の概要...",      // エグゼクティブサマリー
  "trends": ["AI規制", "半導体"],   // トレンドキーワード
  "articles": [
    {
      "title": "ニュースタイトル",
      "summary": "要約テキスト",
      "category": "policy",        // カテゴリID
      "impact": "高",              // 影響度: 高/中/低
      "region": "global",          // 地域: global/domestic
      "source": "情報源名",
      "beginnerNote": "初心者向け解説（任意）",
      "impactNote": "影響解説（任意）"
    }
  ],
  "ideas": [
    {
      "title": "活用アイデア",
      "icon": "🚀",
      "desc": "詳細説明",
      "color": "#2563EB",
      "tags": ["タグ1", "タグ2"]
    }
  ]
}
```

### ローカルストレージ構造
- **ainews_gas_url**: GAS デプロイURL
- **ainews_index**: 保存済み日付一覧 ["2026-05-04", "2026-05-03", ...]
- **ainews_20260504**: 日付別データ (date.replace(/-/g,"")でキー生成)

- **高** (#DC2626): 業界に大きな影響
- **中** (#D97706): 注目すべき動向

### 地域分類
- **global**: 世界的なニュース
- **domestic**: 日本国内のニュース

## 📱 機能仕様

### タブ構成
1. **📰 ニュース** (news)
   - エグゼクティブサマリー

2. **🌍 世界情勢** (global)
   - グローバルニュースのみ表示
   - 影響度高の統計表示

3. **🇯🇵 国内情勢** (domestic)
   - 国内ニュースのみ表示
   - 影響度高の統計表示

4. **💡 活用アイデア** (ideas)
   - 2カラムグリッド表示
   - タップで詳細展開

5. **📅 過去データ** (history)
   - 保存済み日付一覧
   - 統計サマリー（記事数、影響度高、アイデア数）
   - GAS URL設定

### 記事表示機能
- **カードスタイル**: 左枠色分け（カテゴリ別）
- **詳細展開**: タップでサマリー、解説表示
- **バッジ表示**: 影響度、カテゴリ、情報源
- **ノート機能**: 初心者向け解説、影響解説

### フィルタリング
- **カテゴリ別**: 8つのカテゴリで絞り込み
- **地域別**: グローバル/国内で分離表示
- **影響度**: 高影響度の件数表示

## 🔌 API仕様

### GAS エンドポイント

#### 1. 日付一覧取得
```
GET {GAS_URL}?action=list
Response: { "dates": [{"date": "2026-05-04"}, ...] }
```

#### 2. データ読み込み
```
GET {GAS_URL}?action=load&date=2026-05-04
Response: { "data": { ニュースデータ } }
```

#### 3. データ保存
```
POST {GAS_URL}
Content-Type: text/plain
Body: {
  "action": "save",
  "overwrite": boolean,
  "data": { ニュースデータ }
}
Response: { "success": true }
```

### エラーハンドリング
- **GAS未設定**: ローカルストレージのみで動作
- **ネットワークエラー**: キャッシュデータで継続
- **JSON解析エラー**: ユーザーに分かりやすいエラー表示

## 💾 ストレージ仕様

### localStorage Keys
- `ainews_gas_url`: GAS デプロイURL
- `ainews_index`: 保存済み日付インデックス
- `ainews_YYYYMMDD`: 各日付のデータ

### データ同期
1. **即座保存**: localStorage に即座に保存
2. **背景同期**: GAS への保存は非同期実行
3. **競合回避**: 同じ日付の上書き確認ダイアログ

### キャッシュ戦略
- **優先順位**: localStorage → GAS
- **表示**: キャッシュから即座表示
- **同期**: 背景でGASと同期

## ⚙️ 設定項目

### GAS URL設定
- **設定画面**: モーダル形式
- **検証**: URL形式チェック
- **状態表示**: 連携中/ローカルのみ
- **解除機能**: 連携解除ボタン

### 表示設定
- **統計チップ**: 記事数、影響度高、保存件数
- **メッセージ**: 成功/警告/エラー表示

## 🔄 操作フロー

### 新規データ追加
1. **JSONを貼り付け** ボタンをタップ
2. Claude生成のJSONをテキストエリアに貼り付け
3. プレビューで内容確認
4. 「保存する」ボタンで保存実行
5. 重複時は上書き確認ダイアログ表示

### データ閲覧
1. **過去データ** タブで日付選択
2. 自動でニュースタブに切り替え
3. カテゴリフィルターで絞り込み
4. 記事タップで詳細展開

### Sheets連携設定
1. **Sheets設定** ボタンをタップ
2. GAS デプロイURLを入力
3. 「保存して接続」で連携開始
4. 背景で既存データと同期
